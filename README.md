## 🧑‍💻 Getting Started

### Prerequisites

* Node.js (v18+)
* MongoDB installed or cloud DB URI

### 1. Clone the repository

```bash
git clone https://github.com/B-6219/spotify-fullstack.git
cd spotify-fullstack
```

### 2. Setup backend

```bash
cd backend
npm install
npm run dev
```

### 3. Setup frontend

```bash
cd frontend
npm install
npm run dev
```

> Access the app at `http://localhost:5173/`
